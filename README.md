- **Encryption & Decryption**: Toggle between encoding and decoding text
- **Customizable Settings**:
  - Adjustable shift value
  - Modifiable modulo value
  - Customizable alphabet
  - Case sensitivity options (maintain, lowercase, uppercase)
  - Foreign character handling (remove or ignore)
- **Real-time Processing**: Instant results as you type

- **Responsive Design**: Works on desktop and mobile devices
